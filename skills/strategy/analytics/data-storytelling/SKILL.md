# Data Storytelling

Transform raw data into compelling narratives that drive decisions and inspire action. Combine data evidence, narrative meaning, and visual clarity. Use when presenting analytics to executives, creating quarterly reviews, building investor presentations, or communicating insights to non-technical audiences.

## When to Use

- Presenting analytics to executives or board
- Creating quarterly business reviews
- Building investor presentations
- Writing data-driven reports
- Making recommendations based on data analysis

## Story Structure

```
Setup (Context + Baseline) → Conflict (Problem/Opportunity) → Resolution (Insights + Recommendations)
```

### Narrative Arc
1. **Hook:** Grab attention with surprising insight
2. **Context:** Establish the baseline metrics
3. **Rising Action:** Build through supporting data points
4. **Climax:** The key insight that changes understanding
5. **Resolution:** Specific recommendations
6. **Call to Action:** Next steps with resource ask

## Story Frameworks

### Problem-Solution Story
Hook with dollar impact → Context (current metrics vs benchmarks) → Pattern discovery in data → Solution with expected ROI → Specific budget ask.

### Trend Story
Where we started → What changed (timeline) → Before/after comparison table → Key insight → Forward-looking targets.

### Comparison Story
Frame the question → Side-by-side analysis with weighted scoring matrix → Data-backed recommendation → Risk mitigation plan.

## Visualization Techniques

1. **Progressive reveal:** Start simple, add layers across slides
2. **Contrast and compare:** Before/after, This/That side-by-side panels
3. **Annotation and highlight:** Call out key events on trend lines, shade growth periods, add threshold lines

## Writing Techniques

### Headlines That Work
Bad: "Q4 Sales Analysis" → Good: "Q4 Sales Beat Target by 23% — Here's Why"
Formula: [Specific Number] + [Business Impact] + [Actionable Context]

### Handling Uncertainty
Acknowledge limitations with confidence intervals. Present ranges ("Impact estimate: $400K-$600K"). State assumptions clearly. Never guarantee specific results.

## Best Practices

**Do:** Start with the "so what", use rule of three, show don't tell, connect to audience goals, end with clear action.
**Do Not:** Data dump without curation, bury the insight, use jargon without explanation, show methodology before context, present numbers without narrative meaning.
