Guide illustration and icon design for brand content. Define style, provide direction, and review output.

## When to Use
- Custom icons for a feature set
- Editorial illustrations for blog/content
- Brand-specific visual elements

## Direction Format
1. Style reference (flat, isometric, hand-drawn, geometric)
2. Color palette (from brand tokens)
3. Subject matter and composition
4. Technical specs (SVG, PNG size, viewBox)
5. Accessibility (sufficient contrast, alt text)
